Projetos em Ciência de Dados

Ordenação de lista de clientes para contato.

Este projeto foi realizado para uma empresa que fornece seguro de saúde e está estudando a possibilidade de oferecer aos seus segurados um novo produto: seguro de automóveis.

Pontos interessantes de destaque:

O problema de negócio envolveu a técnica de "learn-to-rank", que faz parte dos problemas de classificação. Foi possível gerar valor para a empresa através da priorização do contato com clientes que mostraram ter maior potencial em obter o seguro, evitando o desperdício de recursos com clientes de baixo potencial.

Projeto para uma rede de lojas que teve como a previsão de vendas para as seis semanas seguintes. Foram utilizadas ferramentas de bibliotecas como Matplotlib, Seaborn, Numpy, Sci-kit Learn.
